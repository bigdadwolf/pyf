chapter16 逻辑和位操作

  • 逻辑指令
    • and
      • + - 清除某一位或者某几位
        • AL =  11010110
        • BL  =  11111100
        • and AL, BL = 11010100
        • 将AL最右边2位清除，即设置为0
      • + - 隔离某些位
        • and  AL, 1
        • jz  even_number
        • 判断AL最右边1位是否为1
    • + - or
      • 设置某一位或者某几位
        • AL = 11010110
        • BL = 00000011
        • or AL, BL = 11010111
        • 将AL最右边2位置位，即设置为1
      • 剪切和粘贴
        • and AL,55H ; cut odd bits
        • and BL,0AAH ;  cut even bits
        • or  AL, BL   ; paste them together
    • + - not
      • + - 求补码
        • not AL
        • inc AL
      • + - 专门的补码指令
        • neg
    • + - xor
      • 翻转某一位或者某几位
        • xor EAX, 0FH
        • 将EAX中低4位翻转
      • 初始化寄存器为0
        • xor EAX, EAX
        • EAX = 0
    • + - test
      • 执行逻辑and操作
      • test不改变目标操作数，而and则改变目标操作数
  • 移位指令
    • 逻辑移位
      • + - shl
        • doubling a unsigned number
      • + - shr
        • halving a unsigned number
      • 以上2个指令工作于无符号数
    • 算数移位
      • + - sar
        • halving a signed number
      • + - sal
        • doubling a signed number
      • 以上2个指令工作于有符号数
    • 双移位操作
      • shld
        • shld dest,src,count
      • shrd
        • shrd dest,src,count
      • dest必须是寄存器或者内存
      • src必须是寄存器
  • + - 循环移位指令
    • 带CF的循环移位指令
      • rcl
      • rcr
    • 不带CF的循环移位指令
      • ror
      • rol
  • 位指令
    • 位测试和修改指令
      • + - bt
        • bit test, 将选择的位拷贝到CF中
      • + - bts
        • bit test and set, 将选择的位拷贝到CF中，并将该位设置为1
      • + - btr
        • bit test and reset，将选择的位拷贝到CF中，并将该位设置为0
      • + - btc
        • bit test and complement, 将选择的位拷贝到CF中，并将该位取反
      • 以上4个指令将选择的位拷贝到CF中
    • 位扫描指令
      • + - bsf
        • bsf dest_reg, operand; bit scan forward
        • 从右到左扫描，找到第一个1出现的位置
      • + - bsr
        • bsf dest_reg, operand; bit scan reverse
        • 从左到右扫描，找到第一个1出现的位置
